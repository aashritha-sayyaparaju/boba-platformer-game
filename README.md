# cyoa-game
BOBA PLATFORMER GAME
A fun 2D platformer game where you play as a Thai tea character collecting tapioca pearls!

Game Overview:
Jump across platforms and collect all the tapioca pearls to win! Your Thai tea character can double jump to reach higher platforms and gather delicious boba treats.

Features:
Double Jump Mechanics - Jump twice to reach high platforms
Collectible Boba balls - Gather all the boba to complete the game
Scrolling Camera - The world moves with your character
Game Over Screen - Try again button to restart from the beginnning
Score Tracking - See how many boba pearls you've collected

How to Play:
Use arrow keys to move your character
Press up arrow twice for double jump

Goal:
Collect all 5 boba balls to win!
Avoid falling off the platforms

How to Run:
Make sure you have Python installed
Install pygame: pip install pygame
Run the game: python cyoa-game.py
